
# BrightTV Viewership Analytics

## Problem Statement

BrightTV aims to grow its subscription base within the current financial year. However, limited insights into user behavior, content preferences, and viewing patterns make it challenging for the Customer Value Management (CVM) team to implement effective growth strategies.


## Aim of the Project

To analyze user profiles and viewership data in order to generate actionable insights that support increased content consumption and subscriber growth.

## Objectives

* Clean and prepare the dataset (including converting UTC timestamps to South African time).
* Analyze user demographics (age, gender, and location).
* Identify viewership trends across time slots and sessions.
* Determine factors influencing content consumption.
* Highlight top-performing content and underperforming periods.
* Develop data-driven recommendations to increase engagement and grow the user base.

## Summary of Results

* Viewership varies significantly by **time of day**, with peak consumption occurring during specific time slots.
* Certain **content types and shows** consistently attract higher engagement.
* **Demographic factors** (age and gender) influence viewing preferences and behavior.
* Lower consumption periods present opportunities for targeted content strategies and promotions.
* Insights suggest that personalized recommendations and strategic content scheduling can improve overall engagement and subscriptions.

## Tools Used

* **Power BI** – Data visualization and dashboard creation
* **SQL** – Data extraction, transformation, and querying
* **Excel** – Data cleaning and preprocessing
